This project provides a robust solution for understanding audience sentiment and engagement on YouTube. By leveraging the YouTube Data API and Natural Language Processing (NLP) techniques, this tool fetches comments from a specified video URL, processes the raw text, and classifies the expressed opinions into positive, negative, or neutral categories.

Key Goals
Audience Insight: Help content creators and marketers quickly gauge viewer reception to specific videos.

Scalable Analysis: Handle thousands of comments efficiently using Python and data science libraries.

Visualization: Provide clear, interactive visualizations of sentiment trends and keyword distribution.

Comment Extraction: Uses the official YouTube Data API (v3) for reliable data retrieval.

Sentiment Analysis: Utilizes a lexicon-based model (e.g., VADER) or a machine learning classifier (e.g., Logistic Regression) for sentiment scoring.
